### **📡 Available Endpoints**
```
✅ GET  /api/health          - Server health check

✅ GET  /api/posts           - Posts with author info
✅ GET  /api/posts/:id       - Individual post details
✅ GET  /api/posts/:id/comments - Post comments & replies

✅ POST /api/posts/:id/comments - Add comments
```
